## 15. Open Source Telemetry

The open source version of Aegis includes optional, anonymous telemetry to help improve the product.

**What's collected:**
- Aegis version
- Operating system
- Feature usage (anonymous counts)
- Error reports

**What's NOT collected:**
- Personal information
- Query contents
- Database contents
- IP addresses (for open source)

**Opt-out:**
```toml
[telemetry]
enabled = false
```

Or set environment variable:
```bash
export AEGIS_TELEMETRY=false
```
